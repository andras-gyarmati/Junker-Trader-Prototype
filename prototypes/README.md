# Prototypes

Each prototype is self-contained in its own subfolder:

- `roadtrip-trader`: current trading + travel run loop
- `classic-trader`: old trader-only loop
- `backpack-hitchhike`: new travel-only concept
- `time-loop-platformer`: tiny rewind + echo 2D platformer mechanic test

When creating new concepts, add a new sibling folder here with its own `index.html`, `README.md`, and scripts.
