Time-Loop Echo Platformer (2-Character Timeline)

Goal: get both characters into the green exit in the same timeline state.

Controls

A/D or Left/Right: move
Space: jump
E: ability (throw charge+launch / rope place)
Q or Tab: switch controlled character anytime
Tap R: instant rewind 2 seconds + branch
Hold R: visual rewind playback at 2x; release to branch
Enter: full reset

Xbox: Left Stick/DPad move, A jump, X ability, Y rewind (tap/hold), LB/RB switch, Menu reset

Roles

Thrower: ability charges, then triggers a short upward-launch pulse (orange build-up).
Rope: ability places rope on marked anchors; touching rope instantly climbs.

Debug

Timeline

Markers: ▲ jump, ■ throw, ● rope
