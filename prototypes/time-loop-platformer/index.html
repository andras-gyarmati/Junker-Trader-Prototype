Time-Loop Echo Platformer (2-Character Timeline)

Goal: get both characters into the green exit in the same timeline state.

Controls

A/D or Left/Right: move
Space: jump
E: ability (throw window / rope place / rope climb)
Q or Tab: switch controlled character anytime
Hold R: rewind world backward
Release R: branch (truncate active track, rewrite from now)
Enter: full reset

Roles

Thrower: ability opens throw window (shown as taller orange state).
Rope: ability places rope on marked anchors; on rope, press E to instantly climb.
